Live Demo: [View Payoo Live](https://ruhit1000.github.io/Payoo-Mobile-Banking-Project/)

Payoo is a fintech-inspired web application built to master JavaScript DOM Manipulation and real-time state management. This project simulates core banking features like secure login, balance tracking, and transaction logic within a mobile-first interface.

üöÄ Core Features
Authentication Logic: Simulated secure login with credential validation.

Real-time Transactions: Functional "Add Money" and "Cash Out" features that update the main balance instantly without page refreshes.

Form Validation: Handles user input errors to ensure only valid amounts are processed.

Seamless UI Navigation: Uses a single-page approach (SPA-lite) to switch between different banking sections.

Mobile-First Design: Fully responsive layout optimized for mobile banking experiences.
